## 1. HY-Embodied-0.5: Embodied Foundation Models for Real-World Agents

**所属方向：** Embodied AI

**arXiv 链接：** https://arxiv.org/abs/2604.07430

**中文摘要：**
HY-Embodied-0.5 提出了一个面向具身智能体的双变体基础模型家族，采用 2B 激活参数的 MoT 架构，结合原生分辨率 ViT 和视觉潜在令牌，以增强细粒度视觉感知与模态感知推理能力。通过迭代强化学习与拒绝采样 SFT 构成的自进化后训练循环，显著提升了具身预测与规划性能，并通过在线策略蒸馏将 32B 模型能力迁移至 2B 变体。2B 模型在 16 个基准测试上超越当前最优方法，32B 变体性能与 Gemini 3.0 Pro 相当，并通过 VLA 机器人控制完成了真实世界验证。

**主要贡献：**
- 提出 MoT 双变体架构，实现高效细粒度具身感知
- 构建自进化后训练闭环（迭代 RL + 拒绝采样 SFT）
- 在线策略蒸馏实现大小模型能力对齐，并在真实机器人上完成部署验证

## 2. Large Language Model Post-Training: A Unified View of Off-Policy and On-Policy Learning

**所属方向：** 大语言模型（LLM）/ 强化学习（RL）

**arXiv 链接：** https://arxiv.org/abs/2604.07941

**中文摘要：**
本文通过"轨迹来源"视角——离策略（外部轨迹）与在策略（学习者生成的 rollout）——对大语言模型后训练进行统一框架化，并引入三种行为功能角色：有效支持扩展、策略重塑与行为巩固。将 SFT、偏好优化、RL、蒸馏及混合流程重新解读为对模型行为的协同干预，而非孤立技术，强调系统性能依赖于跨学习范式、功能角色和训练阶段的协同设计。

**主要贡献：**
- 首次以轨迹来源为核心统一 LLM 后训练全流程
- 提出"行为三角色"框架，为后训练设计提供新分析语言
- 将 SFT/RLHF/蒸馏等技术统一纳入协同系统视角，超越单目标优化范式

## 3. DiffuMask: Diffusion Language Model for Token-level Prompt Pruning

**所属方向：** Diffusion Language Model

**arXiv 链接：** https://arxiv.org/abs/2604.06627

**中文摘要：**
DiffuMask 将提示压缩建模为去噪过程，通过并行迭代掩码预测替代传统顺序式、计算代价高昂的令牌移除方法，实现对多个令牌保留掩码的同步预测。该框架可在一分钟内实现高达 80% 的提示长度压缩，比先前方法快数个数量级，同时在域内、域外及跨模型设置下均能保持推理精度，并支持对内容保留程度的可调控制。

**主要贡献：**
- 首次将扩散语言模型范式引入令牌级提示剪枝任务
- 并行迭代掩码预测大幅提升压缩效率（比传统方法快数量级）
- 跨域、跨模型泛化能力强，支持灵活的内容保留率调节

## 4. The Accountability Horizon: An Impossibility Theorem for Governing Human-Agent Collectives

**所属方向：** Agent / AI 治理

**arXiv 链接：** https://arxiv.org/abs/2604.07778

**中文摘要：**
本文使用结构因果模型对人机集合体进行形式化建模，并证明了一个不可能性定理：一旦智能体自主性超过可计算阈值（"问责地平线"）且交互图包含混合反馈环，就不存在任何单一归因框架能同时满足可归因性、可预见性界限、非空洞性与完备性四项基本治理公理。这一结论揭示了分布式自主体系中问责失效是信息论层面的结构性必然，而非工程缺陷，标志着 AI 治理领域首个形式化不可能性定理的建立。

**主要贡献：**
- 提出 AI 治理领域首个形式化不可能性定理
- 精确刻画"问责地平线"相变条件，具有严格数学支撑
- 从根本上重新定义了多智能体治理问题的理论边界

## 5. Epistemic Robust Offline Reinforcement Learning

**所属方向：** 强化学习（RL）

**arXiv 链接：** https://arxiv.org/abs/2604.07072

**中文摘要：**
ERSAC 提出了一种新型离线强化学习框架，以紧凑的 Q 值不确定性集合替代离散集成，利用认知神经网络（Epinets）直接生成不确定性集合，从而将认知不确定性与随机不确定性有效解耦。该方法避免了重采样操作，在偏置行为策略下比 SAC-N 等集成方法具有更强的鲁棒性与样本效率，并引入了新的基准测试套件用于评估多种行为偏置下的离线 RL 性能。

**主要贡献：**
- 以不确定性集合替代集成，提出更紧凑的认知不确定性建模方式
- Epinets 实现认知与随机不确定性解耦，无需重采样
- 构建多行为偏置离线 RL 新基准，推动安全离线学习评估标准化
