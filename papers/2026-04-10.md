## 📄 1. FastDiSS: Few-step Match Many-step Diffusion Language Model on Sequence-to-Sequence Generation

**所属方向：** Diffusion Language Model

**arXiv 链接：** https://arxiv.org/abs/2604.05551

**中文摘要：**
FastDiSS 针对少步扩散语言模型中的两大核心问题提出解决方案：训练与推理阶段自条件信号不匹配导致的误差累积，以及均匀噪声调度引发的损失饱和问题。通过自条件扰动（SCP）和模型感知噪声缩放（MANS）两项技术，有效缓解了推理漂移并维持学习信号多样性。在六个序列到序列基准测试上，推理速度比标准扩散模型提升最高达 400 倍，同时保持甚至提升了生成质量。

**主要贡献：**
- 提出 SCP 机制，通过在前向过程中引入噪声扰动来对齐训练与推理时的自条件分布
- 提出 MANS 机制，对高置信度 token 动态分配更高噪声，防止平凡去噪
- 在保持生成质量的前提下，实现 400× 推理加速，大幅缩小少步扩散模型的质量-效率差距

## 📄 2. Value Mirror Descent for Reinforcement Learning

**所属方向：** 强化学习（RL）

**arXiv 链接：** https://arxiv.org/abs/2604.06039

**中文摘要：**
本文提出值镜像下降（VMD）方法，将凸优化中的镜像下降范式引入价值迭代框架，在确定性环境下实现线性收敛。在随机生成模型设定下，其变体 SVMD 达到近最优样本复杂度 $\tilde{\mathcal{O}}(|\mathbb{S}||\mathbb{A}|(1-\gamma)^{-3}\epsilon^{-2})$，强凸条件下进一步提升至 $\tilde{\mathcal{O}}(|\mathbb{S}||\mathbb{A}|(1-\gamma)^{-5}\epsilon^{-1})$。关键创新在于迭代过程中生成策略与最优策略之间的 Bregman 散度有界，为离线预训练后的在线持续学习提供了理论保障。

**主要贡献：**
- 首次将镜像下降与价值迭代系统性结合，建立 VMD 统一框架
- 在随机设定下证明近最优样本复杂度，优于现有随机价值迭代方法
- 引入 Bregman 散度有界性质，支持离线到在线的高效策略迁移

## 📄 3. AgentGL: Towards Agentic Graph Learning with LLMs via Reinforcement Learning

**所属方向：** Agent + 强化学习（RL）+ 大语言模型（LLM）

**arXiv 链接：** https://arxiv.org/abs/2604.05846

**中文摘要：**
AgentGL 首次将强化学习引入代理式图学习（AGL）框架，将图推理重新建模为拓扑感知导航与 LLM 推理交织进行的过程。智能体通过图原生工具执行多尺度图探索，并借助搜索约束思维机制平衡探索广度与推理深度。采用图条件课程强化学习策略稳定长程策略学习，在节点分类任务上超越基线最高 17.5%，链路预测任务上提升最高 28.4%。

**主要贡献：**
- 首个基于 RL 的代理式图学习框架，无需逐步监督信号
- 设计图原生工具集与搜索约束思维，实现高效图拓扑导航
- 提出图条件课程 RL 策略，有效解决长程图推理中的训练不稳定问题

## 📄 4. CoEnv: Driving Embodied Multi-Agent Collaboration via Compositional Environment

**所属方向：** Embodied AI

**arXiv 链接：** https://arxiv.org/abs/2604.05484

**中文摘要：**
CoEnv 构建了一个组合式环境，将真实世界与仿真空间统一，使多智能体具身系统能够在虚拟空间中进行认知规划，同时在物理世界中执行动作。通过 VLM 驱动的动作合成支持实时及迭代规划，并借助碰撞检测机制保障仿真到现实迁移的安全性。系统在涉及异构机器人的复杂多臂操作任务中取得了高成功率。

**主要贡献：**
- 提出组合式环境架构，打通虚实空间的认知-执行分离
- 支持高层语言接口与代码轨迹生成两种规划模式
- 实现跨异构机器人的安全 sim-to-real 迁移，具备碰撞感知规划能力

## 📄 5. When Agent Markets Arrive

**所属方向：** Agent

**arXiv 链接：** https://arxiv.org/abs/2604.06688

**中文摘要：**
Diagon 是首个端到端的 AI 代理市场系统，支持异构智能体执行任务委托、谈判与基于声誉的交易。研究揭示了一个反直觉现象：透明度与诚信规范在人类市场中有益，但在代理经济中反而诱发自我隔离与纠纷增加，整体生产力下降。系统性地展示了竞价机制、不完整合同与声誉动态之间的涌现交互如何导致市场脆弱性。

**主要贡献：**
- 构建首个可编程代理市场系统 Diagon，支持异构 AI 代理的完整市场行为模拟
- 发现制度设计（透明度、诚信规范）在代理经济中产生与人类市场截然不同的效果
- 为代理经济的制度设计提供实证依据，警示直接移植人类市场规则的风险
