## 1. 📄 FastDiSS: Few-step Match Many-step Diffusion Language Model on Sequence-to-Sequence Generation

**所属方向：** Diffusion Language Model

**arXiv 链接：** https://arxiv.org/abs/2604.05551

**中文摘要：**
FastDiSS 针对少步扩散语言模型中的两大核心问题——训练与推理时自条件不匹配导致的误差累积，以及均匀噪声调度造成的损失饱和——分别提出了自条件扰动（SCP）和模型感知噪声缩放（MANS）两种解决方案。SCP 通过对自条件信号引入前向过程生成的更嘈杂预测进行扰动，使推理阶段噪声分布更加匹配；MANS 则动态为高置信度 token 分配更高噪声以维持学习信号多样性。该框架在六个序列到序列基准测试上实现了比标准扩散模型快 **400 倍**的推理速度，同时保持甚至提升了生成质量。

**主要贡献：**
- 提出 SCP 机制，有效缓解训练-推理自条件不匹配问题
- 提出 MANS 机制，解决后期训练损失饱和问题
- 在少步设置下大幅缩小质量-效率差距，推理加速达 400×

## 2. 📄 Rectified Schrödinger Bridge Matching for Few-Step Visual Navigation

**所属方向：** Embodied AI

**arXiv 链接：** https://arxiv.org/abs/2604.05673

**中文摘要：**
RSBM 提出了一种在 Schrödinger Bridge（ε=1）与确定性最优传输（ε→0）之间连续插值的统一框架，证明了速度场结构在不同正则化强度下的不变性，使单一网络可处理全部正则化强度设置。理论上，方差随 ε 减小而线性降低，支持稳定的少步（仅 3 步）ODE 积分用于视觉导航任务。实验上，RSBM 在 3 步内达到 94% 余弦相似度和 92% 成功率，相比基线方法 MSE 降低 6.3 倍、函数评估次数减少 3.8 倍，且无需蒸馏或多阶段训练。

**主要贡献：**
- 提供速度场结构不变性的严格理论证明
- 实现高保真生成策略与实时具身控制的统一
- 无蒸馏情况下以极少步数达到 SOTA 导航性能

## 3. 📄 AgentGL: Towards Agentic Graph Learning with LLMs via Reinforcement Learning

**所属方向：** Agent + 强化学习（RL）+ 大语言模型（LLM）

**arXiv 链接：** https://arxiv.org/abs/2604.05846

**中文摘要：**
AgentGL 是首个将强化学习引入 Agentic Graph Learning（AGL）的框架，将图推理重新建模为拓扑感知导航与 LLM 推理交替进行的过程。框架配备图原生工具支持多尺度图探索，通过搜索约束思维平衡探索深度与推理宽度，并利用图条件课程 RL 策略在无逐步监督的情况下稳定长时域策略学习。在节点分类任务上提升最高达 **17.5%**，链路预测任务上提升最高达 **28.4%**，显著超越现有 GraphLLM 和 GraphRAG 基线。

**主要贡献：**
- 首次将 RL 课程学习框架应用于 Agentic Graph Learning
- 提出搜索约束思维机制，兼顾探索与推理效率
- 在多个图学习任务上实现大幅性能提升

## 4. 📄 Value Mirror Descent for Reinforcement Learning

**所属方向：** 强化学习（RL）

**arXiv 链接：** https://arxiv.org/abs/2604.06039

**中文摘要：**
本文提出 Value Mirror Descent（VMD），将凸优化中的镜像下降方法融入值迭代框架。在确定性设置下 VMD 实现线性收敛；其随机变体 SVMD 在生成模型下达到近最优样本复杂度 $\tilde{\mathcal{O}}(|\mathbb{S}||\mathbb{A}|(1-\gamma)^{-3}\epsilon^{-2})$，强凸条件下进一步改善至 $\tilde{\mathcal{O}}(|\mathbb{S}||\mathbb{A}|(1-\gamma)^{-5}\epsilon^{-1})$。关键创新在于整个迭代过程中生成策略与最优策略之间的 Bregman 散度有界，为离线训练后的在线延续提供了理论保障。

**主要贡献：**
- 将镜像下降引入值迭代，建立严格的收敛理论
- 在随机设置下达到近最优样本复杂度
- 证明迭代过程中 Bregman 散度有界这一先前方法所缺乏的关键性质

## 5. 📄 Context-Value-Action Architecture for Value-Driven Large Language Model Agents

**所属方向：** 大语言模型（LLM）+ Agent

**arXiv 链接：** https://arxiv.org/abs/2604.05939

**中文摘要：**
本文提出 Context-Value-Action（CVA）架构，通过将行动生成与认知推理解耦来解决 LLM Agent 行为僵化问题。该架构以刺激-有机体-反应模型和 Schwartz 基本人类价值理论为基础，使用在真实人类行为数据上训练的 Value Verifier 对动态价值激活建模，从而实现更真实、多样的类人行为。在包含超过 110 万条真实世界交互轨迹的 CVABench 上的实验表明，纯提示驱动推理会加剧价值极化，而 CVA 有效缓解了这一问题并实现更高的行为保真度。

**主要贡献：**
- 提出首个基于人类价值理论的 LLM Agent 行为建模架构
- 构建含 110 万+真实交互轨迹的 CVABench 评测基准
- 揭示并缓解提示驱动推理导致价值极化的现象
