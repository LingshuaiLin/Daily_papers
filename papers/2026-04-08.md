## 📄 1. LLaDA-MoE: A Sparse MoE Diffusion Language Model

**所属方向：** Diffusion Language Model

**arXiv 链接：** https://arxiv.org/abs/2509.24389

**摘要：**
LLaDA-MoE 将稀疏混合专家（MoE）架构引入掩码扩散语言模型（MDM），在推理时仅激活 1.4B 参数，却在知识、推理、编程和 Agent 任务上达到最优性能，显著优于此前需要 8B 参数的稠密扩散模型。该模型从头在 20T token 上训练，超越了 LLaDA、LLaDA 1.5 和 Dream 等基线，并在指令跟随能力上媲美 Qwen2.5-3B-Instruct。通过专家路由和负载均衡机制，MoE 集成在保持高效推理的同时未损失性能增益。

**主要贡献：**
- 首次将稀疏 MoE 架构成功引入掩码扩散语言模型，实现参数高效推理
- 在多个基准任务上树立新的 SOTA，证明扩散模型可与自回归模型竞争
- 为扩散语言模型的规模化提供了可行的工程路径

## 📄 2. Continuous Diffusion Model for Language Modeling

**所属方向：** Diffusion Language Model

**arXiv 链接：** https://arxiv.org/abs/2502.11564

**摘要：**
RDLM 通过将离散类别数据嵌入统计流形的几何结构，提出了一种用于语言建模的连续扩散框架，利用微分同胚映射将其对应到超球面的正象限。该方法在 Fisher-Rao 流形上建立了离散扩散过程与连续流之间的直接联系，并利用径向对称性实现无模拟训练。在语言、图像和生物序列任务上，RDLM 性能与自回归模型相当，并优于此前的连续扩散方法。

**主要贡献：**
- 从黎曼几何角度统一了离散与连续扩散过程，理论贡献深厚
- 利用流形上的几何结构实现无模拟训练，大幅降低训练复杂度
- 在多模态任务（语言、图像、生物序列）上展示了方法的通用性

## 📄 3. A Call for Embodied AI

**所属方向：** Embodied AI

**arXiv 链接：** https://arxiv.org/abs/2402.03824

**摘要：**
本文将具身 AI（E-AI）定位为通向通用人工智能（AGI）的必要基础，深刻批判了静态数据驱动的大语言模型在因果推理、感知运动耦合和动态信念修正方面的根本局限。E-AI 以具身认知和主动推断为理论基础，强调智能体需通过与物理环境的持续交互来构建和修正知识。论文指出当前 LLM 缺乏内在动机和真实世界接地，认为真正的智能必须来源于动态的身体-环境交互。

**主要贡献：**
- 系统性地论证了具身 AI 相对于纯 LLM 路径的认知科学必要性
- 提出了"对真理的关切"这一具身认知核心概念，为 AGI 研究指明方向
- 为具身 AI 研究社区提供了清晰的哲学与技术框架

## 📄 4. The AI Agent Index

**所属方向：** Agent

**arXiv 链接：** https://arxiv.org/abs/2502.01635

**摘要：**
AI Agent Index 构建了首个已部署 Agent AI 系统的公开数据库，系统记录了各系统的技术组件（推理、规划、记忆、工具使用等）、应用领域及安全实践。研究发现，开发者虽乐于公开能力和用例，但安全披露极度匮乏——正式安全政策披露率仅 19.4%，第三方评估参与率不足 10%。这一发现揭示了实际部署中透明度与风险管理存在的严重缺口。

**主要贡献：**
- 建立了首个系统性的 Agent AI 部署数据库，为该领域提供重要的实证基础
- 量化揭示了 Agent 系统在安全披露方面的严峻现状，具有重要政策参考价值
- 为 Agent 评估标准化和治理框架的建立提供了数据支撑

## 📄 5. AI Agent Behavioral Science

**所属方向：** Agent

**arXiv 链接：** https://arxiv.org/abs/2506.06366

**摘要：**
本文提出"AI Agent 行为科学"这一新范式，将研究焦点从模型内部结构转向 Agent 在动态环境中可观测的、上下文依赖的行为。论文系统化了跨个体、多 Agent 及人机交互的实证研究，识别出内在特征、环境约束和反馈回路等关键行为维度，并揭示了谈判、欺骗等涌现行为。该框架将公平性、安全性等 AI 治理原则重新定义为动态、上下文敏感的行为属性，而非静态模型特征。

**主要贡献：**
- 开创性地将行为科学方法论引入 AI Agent 研究，提供全新分析视角
- 系统梳理了个体、多 Agent、人机交互三个层次的行为研究框架
- 为负责任 AI 的动态化、情境化评估与治理提供了科学基础

## Quick Picks

### 🌟 最值得深读的论文

#### 首选：**LLaDA-MoE: A Sparse MoE Diffusion Language Model**（2509.24389）

**推荐理由：** 这是扩散语言模型走向实用化的重要里程碑。将 MoE 架构引入 MDM 不仅是工程上的创新，更解决了扩散模型长期面临的推理效率瓶颈。以 1.4B 激活参数超越 8B 稠密模型的结果极具说服力，且在 Agent 任务上的优异表现意味着其实用价值不局限于语言生成本身。对于关注非自回归生成模型未来走向的研究者，此文不可错过。

#### 次选：**The AI Agent Index**（2502.01635）

**推荐理由：** 在 Agent 技术快速落地部署的当下，这篇论文提供了极为稀缺的实证数据视角。它不追求技术突破，而是通过系统性调查揭示了整个行业在安全透明度上的集体性缺失——这对研究者、开发者和监管者都有直接的参考意义。数据库本身也是宝贵的研究资源，适合所有关注 AI 治理与负责任部署的读者深入研读。
